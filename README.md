## ☕ Coffee Shop

**Сайт кофейни с каталогом, заказом и админ-панелью.**  
Полнофункциональное веб-приложение на PHP + MySQL. Адаптивный дизайн, авторизация пользователей, управление меню через админку.

Для того, чтобы скопировать сайт, необходимо скачать архив, распаковать в папку domains для Open Server и импортировать базу данных.
